# Face Emotion Detection using CNN

This project is a **real-time face emotion detection system** built using a **Convolutional Neural Network (CNN)** with **PyTorch** and **OpenCV**.  
It detects human faces from a webcam feed and predicts the corresponding facial emotion.

---

## Features
- Real-time face detection using Haar Cascade
- Emotion classification using a CNN model
- Webcam-based live emotion prediction
- Supports multiple emotion classes
- Simple and modular code structure

---

## Model Details
- Framework: PyTorch
- Model Type: Convolutional Neural Network (CNN)
- Input: Grayscale face images
- Output: Emotion class label

---

## Tech Stack
- Python
- PyTorch
- OpenCV
- NumPy
